# THƯ MỤC CHỨA MÔ HÌNH AI (YOLO)

Thư mục này dùng để lưu trữ các file mô hình phát hiện biển số xe định dạng `.pt` của YOLO.

### Hướng dẫn:
1. Đặt file mô hình nhận diện biển số đã train của bạn tên là **`best.pt`** vào thư mục này để hệ thống tự động load khi khởi chạy.
2. Nếu không có file mô hình custom, hệ thống sẽ tự động tải file **`yolov8n.pt`** (YOLOv8 Nano mặc định) về thư mục này để chạy thử nghiệm.
